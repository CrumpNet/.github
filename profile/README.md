# CrumpNet

CrumpNet is the personal homelab and infrastructure project of Joshua Crump — an IT infrastructure engineer documenting what gets built, what breaks, and what gets learned from both.

## What This Is

CrumpNet is a homelab built around real infrastructure work. The goal is not to run every possible tool, but to build a useful, maintainable environment that supports learning, testing, and documentation outside of production.

The lab covers:

- **Identity** — Active Directory, DNS, DHCP, and PKI
- **Visibility** — Monitoring, alerting, and security event collection
- **Services** — Docker workloads, Linux administration, and backup
- **Networking** — VLANs, segmentation, and firewall management
- **Automation** — PowerShell scripting and configuration management

## The Site

Technical writeups, design decisions, and lessons learned are published at **[crump.net](https://www.crump.net)**.

The writing is meant to be useful — to come back to during an outage, share with a colleague, or revisit years later without having to reverse-engineer past decisions.
